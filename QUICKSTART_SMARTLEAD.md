# 🚀 Smartlead Dashboard - Швидкий Старт

## ⏱️ 3 хвилини до запуску

### Крок 1: Встановіть залежності (30 сек)

**Linux/Mac:**
```bash
chmod +x run_smartlead.sh
./run_smartlead.sh
```

**Windows:**
```
Подвійний клік на run_smartlead.bat
```

**Або вручну:**
```bash
pip install -r requirements_smartlead.txt
```

---

### Крок 2: Налаштуйте API ключ (1 хв)

1. Відкрийте `smartlead_config.py`
2. Знайдіть рядок:
   ```python
   API_KEY = "YOUR_API_KEY_HERE"
   ```
3. Замініть на ваш реальний ключ:
   ```python
   API_KEY = "782e6baa-a185-4779-9a05-799e4c21a65d_zduejn7"
   ```
4. Збережіть файл

**Де взяти API ключ:**
- Smartlead.ai → Settings → API → Copy Key

---

### Крок 3: Запустіть дашборд (10 сек)

```bash
streamlit run smartlead_dashboard_v2.py
```

Відкриється браузер на `http://localhost:8501`

## 🎯 Що далі?

### Експлоруйте дашборд:

1. **📊 Вкладка "Огляд"**
   - Подивіться воронку конверсії
   - Перевірте розподіл відповідей

2. **📈 Вкладка "Детальна статистика"**
   - Знайдіть топ кампанії
   - Проаналізуйте кореляції

3. **🎯 Вкладка "Advanced Analytics"**
   - Теплова карта продуктивності
   - Radar chart порівняння

4. **📋 Вкладка "Таблиця"**
   - Відфільтруйте дані
   - Експортуйте в CSV

## 🛠️ Проблеми?

### ❌ "API_KEY не налаштовано"
→ Перевірте крок 2, переконайтесь що замінили ключ

### ❌ "Module not found"
→ Запустіть: `pip install -r requirements_smartlead.txt`

### ❌ "Streamlit not found"
→ Запустіть: `pip install streamlit`

### ❌ Дані не завантажуються
→ Перевірте:
1. Інтернет з'єднання
2. API ключ коректний
3. Є активні кампанії в Smartlead
