```

       │
       ↓

│  smartlead_config.py     │

│  • BASE_URL              │
│  • get_headers()         │
└──────┬───────────────────┘
       │
       ↓
┌──────────────────────────┐
│  api_request()           │
│  • GET /campaigns        │
│  • GET /analytics        │

│  • GET /email-accounts   │

       │
       ↓
┌──────────────────────────┐
│  Smartlead API           │

       │
       ↓
┌──────────────────────────┐
│  Response Processing     │
│  • JSON parsing          │
│  • Error handling        │
│  • Rate limiting         │
└──────┬───────────────────┘
       │
       ↓
┌──────────────────────────┐

       ↓
```

## 🎓 Навчальний Шлях

### Новачок:
```
1. Прочитайте: QUICKSTART_SMARTLEAD.md (5 хв)
2. Встановіть: requirements_smartlead.txt (2 хв)
3. Налаштуйте: smartlead_config.py (2 хв)
4. Перевірте: check_installation.py (1 хв)
5. Запустіть: smartlead_dashboard_v2.py
6. Експлоруйте: вкладки дашборду (15 хв)
```

### Досвідчений:
```
1. Огляньте: SMARTLEAD_README.md
2. Налаштуйте: smartlead_config.py (advanced options)
3. Запустіть: dashboard_v2.py з custom параметрами
4. Експортуйте: smartlead_data_export.py
5. Інтегруйте: в ваш workflow
```

### Експерт:
```
1. Модифікуйте: скрипти під ваші потреби
2. Додайте: нові API endpoints
3. Створіть: custom візуалізації
4. Автоматизуйте: cron jobs / scheduled tasks
5. Інтегруйте: в ваш data pipeline
```

## 🎯 Швидкий Доступ

### Команди Terminal:

```bash
# Перевірка
python check_installation.py

# Запуск дашборду
streamlit run smartlead_dashboard_v2.py

# Експорт даних
python smartlead_data_export.py

# Валідація конфігурації
python smartlead_config.py

# Перегляд документації
cat QUICKSTART_SMARTLEAD.md
```

### Які файли редагувати:

| Що змінити | Файл |
|------------|------|
| API ключ | `smartlead_config.py` |
| Ліміти/timeouts | `smartlead_config.py` |
| Візуалізації | `smartlead_dashboard_v2.py` |
