# 📊 Smartlead Advanced Dashboard

Розширений дашборд для аналізу даних з Smartlead.ai з детальною візуалізацією та статистикою.

## 🚀 Можливості

### Основні функції:
- ✅ **Загальна статистика** - огляд всіх кампаній
- 📊 **Детальна аналітика кампаній** - метрики відкриттів, кліків, відповідей
- 🎯 **Воронка конверсії** - візуалізація шляху користувача
- 📈 **Порівняльний аналіз** - порівняння декількох кампаній
- 📧 **Статус email акаунтів** - моніторинг warmup та лімітів
- 👥 **Перегляд лідів** - детальна інформація по кожній кампанії
- 📥 **Експорт даних** - збереження статистики в CSV

### Візуалізації:
- 📊 Воронка залучення (Sent → Open → Click → Reply)
- 🥧 Діаграми розподілу відповідей (позитивні/негативні/відписки)
- 📈 Топ-5 кампаній по відкриттям та відповідям
- 🎯 Scatter plot кореляції відкриттів та відповідей
- ⚖️ Порівняльні графіки метрик

## 📦 Встановлення

### 1. Встановіть Python 3.8+
```bash
python --version
```

### 2. Встановіть залежності
```bash
pip install -r requirements_smartlead.txt
```

## 🔑 Налаштування API ключа

Відкрийте файл `smartlead_dashboard.py` та вставте ваш API ключ:

```python
RAW_API_KEY = "ВАШ_API_КЛЮЧ_ТУТ"
```

### Як отримати API ключ:
1. Увійдіть в ваш акаунт Smartlead.ai
2. Перейдіть в Settings → API
3. Скопіюйте ваш API ключ
4. Вставте його в скрипт

## ▶️ Запуск дашборду

```bash
streamlit run smartlead_dashboard.py
```

Дашборд автоматично відкриється в браузері за адресою: `http://localhost:8501`

## 🎛️ Використання

### Бічна панель (Sidebar):
- **Авто-оновлення** - автоматичне оновлення даних кожні 5 хвилин
- **Показати детальні ліди** - увімкнути розділ з переглядом лідів
- **Період аналізу** - вибір часового діапазону

### Розділи дашборду:

#### 1️⃣ Загальна Статистика
- Кількість кампаній (всього/активних/призупинених)
- Ключові метрики: відправлено, відкриття, відповіді

#### 2️⃣ Аналітика Кампаній
**Вкладки:**
- **📊 Огляд** - воронка конверсії та розподіл відповідей
- **📈 Детальна статистика** - топ кампанії та кореляції
- **🎯 Порівняння** - вибір кампаній для порівняння метрик
- **📋 Таблиця** - повна таблиця з фільтрами та експортом

#### 3️⃣ Email Акаунти
- Статус акаунтів
- Warmup статус
- Денні ліміти відправки

#### 4️⃣ Детальна інформація по лідам (опціонально)
- Перегляд лідів обраної кампанії
- Email, ім'я, компанія, статус

## 📊 Метрики що збираються

### По кампаніях:
- Відправлено (total_leads_contacted)
- Відкриття (total_unique_open)
- % Відкриттів (open_rate)
- Кліки (total_unique_click)
- % Кліків (click_rate)
- Відповіді (total_replied)
- % Відповідей (reply_rate)
- Позитивні відповіді (total_positive_reply)
- Негативні відповіді (total_negative_reply)
- Відписки (total_unsubscribe)
- Bounce (total_bounced)

### По email акаунтах:
- Email адреса
- Статус
- Warmup статус
- Денна кількість відправлень
- Ліміт відправки

## 🔧 Додаткові налаштування

### Зміна лімітів лідів:
У функції `get_campaign_leads()` змініть параметр `limit`:
```python
get_campaign_leads(campaign_id, limit=100)  # Змініть 100 на потрібне значення
```

### Інтервал авто-оновлення:
В кінці `main()` змініть значення:
```python
time.sleep(300)  # 300 секунд = 5 хвилин
```

## 📥 Експорт даних

Кнопка **"📥 Завантажити CSV"** у вкладці "Таблиця" дозволяє експортувати всі дані кампаній з:
- Назвами файлів за датою/часом
- Застосованими фільтрами
- Всіма метриками

## 🛠️ API Endpoints які використовуються

```
GET /campaigns                           - Всі кампанії
GET /campaigns/{id}/analytics           - Статистика кампанії
GET /campaigns/{id}/leads               - Ліди кампанії
GET /email-accounts                     - Email акаунти
GET /campaigns/stats                    - Глобальна статистика
```

## ⚠️ Обмеження

- Rate limiting: API може мати обмеження на кількість запитів
- Ліди: За замовчуванням показується максимум 100 лідів на кампанію
- Авто-оновлення: Використовуйте обережно для великої кількості кампаній

## 🐛 Troubleshooting

### Помилка 401/403:
- Перевірте правильність API ключа
- Переконайтесь що ключ активний в налаштуваннях Smartlead

### Повільна завантаження:
- Зменшіть кількість кампаній
- Вимкніть "Показати детальні ліди"
- Збільште timeout в `api_request()`

### Немає даних:
- Перевірте що у вас є активні кампанії
- Переконайтесь що API KEY має необхідні права
